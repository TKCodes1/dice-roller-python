# dice-roller-python

A Python project that rolls dice and displays it in ASCII art.

📙Features -
  Allows user to input the amount of dice they want to spin
  Displays ASCII dice
  Gives random dice rolls
  Places dices next to each other

🧠Things I learnt and used  -
  dictionaries and lists
  nested loops
  ASCII art
  Random module

▶️How to run -
  1. Make sure to have Python installed then run:
 
  ```bash
  python main.py
